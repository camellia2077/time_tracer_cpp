### v0.3.16.1
日志生成程序支持通宵的生成

通过硬编码来生成通宵的天数。

如果是通宵，当前天的第一行就不能为起床相关字段。

每年的第一个月的第一天不能为通宵。因为要测试第一个月的第一天能否与上一年的最后一天生成睡眠时间。

第一次通宵天数为1天，之后1天不通宵，第二次通宵天数为2天，之后1天不通宵，第三次通宵为3天，之后1天不通宵。通过命令行来传递是否
生成器就会按照“1天通宵 -> 1天正常 -> 2天通宵 -> 1天正常 -> 3天通宵...”的模式生成测试日志。
开启通宵天数的生成,-n,--nosleep
