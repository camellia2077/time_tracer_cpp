## v0.2.0 - 2026-01-28

### 架构重构
本版本对 `log_generator` 进行了全面的架构重构，采用 Clean Architecture + DDD 风格的分层设计。

#### 新增层级
- **Domain 层** - 核心业务接口和模型
  - `domain/ports/` - 抽象接口 (ILogGenerator, IFileSystem, ITaskExecutor)
  - `domain/model/` - 领域模型 (Config, GenerationContext, RemarkConfig)

- **Application 层** - 用例编排
  - `application/handlers/` - 处理器 (ConfigHandler, WorkflowHandler)
  - `application/factories/` - 工厂接口 (ILogGeneratorFactory)

- **Infrastructure 层** - 技术实现
  - `infrastructure/persistence/` - 持久化 (FileManager, TomlConfigParser)
  - `infrastructure/concurrency/` - 并发 (ThreadPoolExecutor)
  - `infrastructure/factories/` - 工厂实现

#### 模块整合
- CLI 层简化，合并多个小文件
- Generator 模块独立，实现 Domain 接口

### 功能改进
- 输出文件命名改为 ISO 格式：`2024-01.txt` (原为 `01.txt`)

### 文档
- 为每层添加 `README.md` 说明文件
- 新增架构文档 `docs/log_generator/architecture.md`

### 技术细节
- 修复命名空间冲突：`Application::Application` → `Application::App`
- 更新 PCH 包含顺序解决循环依赖
